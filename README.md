# Bookmarque
An online bookstore class project for CSCI4050 spring '21

# Flask installation for your machine:
https://flask.palletsprojects.com/en/1.1.x/installation/#installation

TlDR:
(linux) 
> $ python3 -m venv venv
> 
> $ . venv/bin/activate
> 
> $ pip install Flask

(windows)
> $ py -3 -m venv venv
> 
> \> venv\Scripts\activate
> 
> $ pip install Flask
